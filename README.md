# vue-slider-component

## 使用方式

``` bash

git clone https://github.com/postbird/vue-slider-component.git

# install dependencies
npm install

yarn install

# serve with hot reload at localhost:8080
npm run dev

yarn dev

# build for production with minification
npm run build

yarn build

```
